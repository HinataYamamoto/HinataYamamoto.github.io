"ロボットに見える" is a first-person shooter game set in a primary school, where you destroy enemies like "Hotline Miami" and complete objectives like "PAYDAY 2".

I've always wanted to make a video game that can compete with "PAYDAY 2" which I played for over 2000 hours.
To make this dream come true, I made an FPS game with all the skills I had at that time.
This experience highlighted many of my problems, but at the same time, it gave me a lot of motivation.
